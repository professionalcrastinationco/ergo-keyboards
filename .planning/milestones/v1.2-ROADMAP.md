## Overview

Improve the visual accuracy of the keyboard layout to better match the physical Choc keyboard, starting with thumb cluster horizontal positioning.

## Phases

### Phase 9: Thumb Cluster Horizontal Alignment

**Goal**: Adjust thumb cluster horizontal positioning so inner thumb keys align vertically with specific main columns (Layer Up with T/G/B, Layer Down with Y/H/N)
**Depends on**: v1.1 complete
**Plans**: 1 plan

Plans:
- [x] 09-01: Adjust CSS positioning for thumb clusters and center gap

**Details:**
The current thumb cluster positioning places the clusters too far outward. In the actual Choc layout:
- Left thumb cluster: Layer Up (NavNum) key aligns vertically with the T, G, B column
- Right thumb cluster: TAB/Symbols key aligns vertically with the Y, H, N column

Implementation requires:
1. Moving each thumb cluster group inward toward center
2. Increasing the gap between left and right halves to prevent Space/Enter overlap
3. Maintaining the existing rotation angles from Phase 8.2
